# Configurando ambiente #

**Ferramentas necessarias:**
 - Node.js
 - Gulp
 - Bower

**Instalando as dependências node:**

    `npm install`
    
**Baixando as dependência sdo Front-End:**

    `bower install`

**Rodando o projeto:**

    gulp w
Sera aberto um servidor local com o endereço: `http://localhost:800`
