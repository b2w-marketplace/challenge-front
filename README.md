 # Rodando o projeto
 
Foi criado um build na pasta dist. 
 - Para rodar o build sem problemas:
   - Rode npm install -g serve
   - Depois rode serve -s dist/build
 
Caso queira rodar como dev, basta rodar: npm install e npm start.
