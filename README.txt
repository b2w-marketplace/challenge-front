Requisitos:
    NodeJs, Npm

Para rodar o app:
    - Entre na pasta ui;
    - Rode esse comando pelo terminal do SO:  npm install && npm run dev
    - abra o link:  http://localhost:8080/ - aqui estará a página requisitada a ser feita

Para ver sem rodar o app, sem executar comandos:
    - Entre na pasta ui/dist
    - Abra o index.html no navegador

Deve se comportar da mesma forma em ambos os casos

Comandos interessantes dentro da pasta ui:
    1º npm install - instalar pacotes importantes para o projeto
    2º npm run dev - rodar a aplicação em modo desenvolvedor - depende 1º
    3º npm run build - construir e otimizar o pacote final - depende 1º
